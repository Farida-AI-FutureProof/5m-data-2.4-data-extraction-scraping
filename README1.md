# Web Scraping Assignment (Module 2.4)

This project scrapes hockey team statistics from:
https://www.scrapethissite.com/pages/forms/

## Features
- Uses a while loop to follow the “next page” button.
- No hardcoded page count.
- Extracts team stats into dictionaries.
